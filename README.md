# Automatic Load Frequency Control (ALFC)

This repository contains a Simulink model of a single-area Automatic Load Frequency Control (ALFC) system.

## Features
- Open loop response
- Primary ALFC (Governor control)
- Secondary ALFC (Integral control)

## Software Used
- MATLAB / Simulink
